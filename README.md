# Software-Testing

Problem: Based on mutation operators used at the level of a statement within a method or function, 
mutation-source-code is used. 

Aim: The mutated program needs to be strongly killed by the designed test cases. At least three 
different mutation operators should be used. 

Eclipse IDE was used to run and test the code, while JUnit was used as the testing framework. 

PITest is a tool for performing code changes that may be installed as an eclipse plugin. These links 
provide information on how to install PITest as well as user documentation 

Source code is a JAVA program for a scientific calculator.

Code and Report have been uploaded in the zipped folder. 

[Project_mutation_source_code.zip](https://github.com/mohar-ops/Software-Testing/files/10437624/Project_mutation_source_code.zip)
